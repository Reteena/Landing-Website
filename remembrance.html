Remembrance

context-aware emotionally-intelligent reminiscence therapy

Overview

Remembrance is an emotionally intelligent, context-aware LLM-based memory assistant specifically designed for Alzheimer's patients. The system combines advanced natural language processing with therapeutic principles to create a supportive tool that helps patients maintain connections to their memories and experiences.

Key Features

The platform features a user interface similar to Obsidian, providing an intuitive and familiar experience for both patients and caregivers. This design choice ensures that the technology doesn't add complexity to an already challenging situation.

Remembrance delivers reminiscence therapy in a non-intrusive manner, carefully balancing therapeutic intervention with user autonomy. The system respects the patient's pace and emotional state, adapting its approach based on real-time context and interaction patterns.

Memory Preservation

At the core of Remembrance is a sophisticated memory extraction system that preserves core memories by analyzing conversations using advanced language models. The system doesn't just store information—it understands context, emotional significance, and relational connections between different memories and experiences.

This approach allows the system to help patients recall not just facts, but the emotional and contextual richness of their memories, which is crucial for effective reminiscence therapy.

Therapeutic Approach

Remembrance is built on established principles of reminiscence therapy, which has been shown to improve quality of life, mood, and cognition in people with dementia. By making this therapy more accessible through technology, we aim to extend its benefits to more patients and their families.

Future Directions

We continue to refine Remembrance based on feedback from healthcare professionals, patients, and caregivers. Our goal is to create a tool that genuinely enhances the quality of life for those living with Alzheimer's disease while supporting their care networks.
